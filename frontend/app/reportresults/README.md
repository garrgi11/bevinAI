# Report Results Page

This page displays the three comprehensive reports generated by the AI analysis:

## Features

1. **Three Report Cards** - Displayed horizontally:
   - Functionality Report (Blue theme)
   - Cost Analysis Report (Green theme)
   - Market Analysis Report (Purple theme)

2. **Interactive Display**:
   - Click any card to view the full report
   - Active card is highlighted with glow effect
   - Smooth animations and transitions

3. **Full Report View**:
   - Displays the complete text of the selected report
   - Formatted with proper spacing and readability

4. **Action Buttons**:
   - **Satisfied**: Confirms the reports and returns to home
   - **Not Satisfied**: Allows user to submit new requirements

## Usage

The page is accessed via URL parameter:
```
/reportresults?projectId=123
```

The page will:
1. Fetch reports from the backend API
2. Display loading state while fetching
3. Show all three reports with animations
4. Allow user to review and take action

## API Endpoint

```
GET /api/v1/reports/:projectId
```

Returns:
```json
{
  "success": true,
  "data": {
    "project_name": "Project Name",
    "functionality_report": "...",
    "cost_analysis_report": "...",
    "market_analysis_report": "...",
    "generated_at": "2025-11-09T..."
  }
}
```
